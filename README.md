## Features

- connect SUI wallet
- create deposit
- withdraw deposit
- list of all user deposits




## Run Locally

Clone the project

```bash
  git clone https://github.com/Proadik/timedeposit
```

Go to the project directory

```bash
  cd timedeposit
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

Visit

```bash
  http://localhost:5173/ or get the link from console
```
